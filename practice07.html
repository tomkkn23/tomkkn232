Microsoft マイクロソフト Windows ウィンドウズ ってどんな会社？

Microsoft とは パソコン用OS Windows ウィンドウズ などで知られる世界的なソフトウェアである。

１９７５年に ビル・ゲイツとポール・アレン によって設立され、近年同OSソフトはパソコン市場のおよそ90%のシェアを持つに至っている。


その他にも

ビジネス用アプリケーションソフトである 「Microsoft オフィス (ワード、エクセルなど)」

インターネット接続用ソフトである 「インターネット・エクスプローラー(通称IE)」

ニュースの配信を行っているぽーたるサイトサービスである 「MSN」 などが有名である。

さらに２００１年からは、家庭用ゲーム機である 「Xbox」を発売しており、米を中心にシェアを持っている。
